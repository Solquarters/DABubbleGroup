Testing Functions

Reset Channel Data Deletes all channels and recreates default channels collection.
Reset User Data Resets all user data to default state - Only activate when you are logged in as guest.
Repopulate Channels Deletes all members from all channels and randomly distributes current registered users to existing public channels.
Create messages Resets all messages to default state.
Create thread messages Resets all thread messages to default state.
